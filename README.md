# Cohort Retention Analysis

User Retention Rate and Cohort Analysis project using SQL and Google Sheets.

## Description
Built a cohort retention analysis model comparing organic vs promo user acquisition quality to track user lifetime activity and optimize marketing performance.

## Tools
* SQL (PostgreSQL)
* [Google Sheets Project Link](https://docs.google.com/spreadsheets/d/1es53C_NOR1TMH9HAY2d4KvT&P3jAMxPxPEPupj7eU/edit?usp=sharing) (Pivot Tables, Conditional Formatting)

## My Tasks
* Cleaned and transformed inconsistent date formats across database tables using SQL.
* Grouped users into monthly cohorts and constructed cohort matrices in Google Sheets using pivot tables and conditional formatting.
* Analyzed customer behaviors to highlight retention drop-offs.

## Result
* Achieved a 25% increase in user retention insights by delivering a data-backed recommendation report for marketing budget optimization.
